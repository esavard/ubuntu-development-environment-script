ubuntu-development-environment-script by esavard

The Ubuntu Developement Environment script is a shell script that setup a development environment for Ubuntu and its derivatives (Kubuntu, Xubuntu, etc.) using a GUI dialog (or console, using -n option). Originally, I created that script to keep track of my development environment between each new release of Ubuntu (and because I like to format my hard drive each time). I provide this script with the hope it would be useful to someone else.

Install

To execute the script, open a terminal and enter: chmod 755 ./ubuntu_devenv.sh ./ubuntu_devenv.sh Follow instructions on screen.
